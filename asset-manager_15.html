접근 허용 이메일 관리

로그인을 허용할 이메일을 등록하세요. 목록에 없는 이메일은 Firebase 로그인 후에도 자동으로 차단됩니다. 목록이 비어 있으면 모든 Firebase 사용자가 접근 가능합니다.

💡 Firebase Console → Authentication → Users 에서 미리 계정을 생성해야 로그인할 수 있습니다.

로컬 데이터 → Firebase 마이그레이션

Firebase 연결 완료 후 사용할 수 있습니다. 기존에 이 앱에서 로컬로 저장한 지출내역·요약·자산 데이터를 Firebase로 이전합니다.
